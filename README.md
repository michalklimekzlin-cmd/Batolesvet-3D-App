# Vivere atque FruiT

Webová / PWA hra – svět se 4 týmy (Human, AI, Glyph, World), s meziprostorem pro moduly (Iskry, Shop, WebGuardian).

## Struktura
- `index.html` – vstupní stránka
- `style.css` – vzhled
- `src/` – logika hry (app, engine, teams)
- `manifest.json` + `service-worker.js` – aby šla aplikace nainstalovat
- `assets/icons/` – ikony pro PWA

## Jak spustit
1. nahraj na GitHub Pages
2. otevři na telefonu
3. přidej na plochu
4. zapisuj nápady → ukládá se do localStorage
